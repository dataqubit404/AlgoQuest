# 🎮 AlgoQuest — The DSA Adventure

> Conquer Data Structures & Algorithms through a gamified, step-by-step debugging code editor.

AlgoQuest transforms traditional DSA practice into an immersive game experience.  
Users explore themed worlds, solve challenges, debug code step-by-step, visualize algorithms, and level up — just like a strategy adventure game.

---

## 🌟 Key Highlights

✨ Gamified learning (Candy-Crush style progression)  
✨ Interactive code editor with step debugger  
✨ Real-time algorithm visualization  
✨ Themed worlds for each DSA topic  
✨ XP, ranks, streaks, and achievements  
✨ AI-assisted hints and explanations  
✨ Progress tracking dashboard
